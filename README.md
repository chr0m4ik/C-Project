# CrossPlatformUISimulator

Учебный проект на C#. Показывает все основные паттерны проектирования (GoF) на примере простого UI-фреймворка.

## Как запустить

1. Открыть `CrossPlatformUISimulator.sln` в Visual Studio / Rider, либо через консоль:
   - `dotnet run --project src/CrossPlatformUISimulator.csproj` — запустить программу
   - `dotnet test tests/CrossPlatformUISimulator.Tests.csproj` — запустить тесты

## Структура

Папка `src/Patterns` — каждый паттерн в своей папке (с номером по порядку изучения).
Папка `tests` — простые тесты на xUnit, по одному файлу на группу паттернов.

Все объяснения, что и где работает — в чате с описанием после кода.
